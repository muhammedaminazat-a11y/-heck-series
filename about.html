О проекте Series Tracker

Series Tracker — это веб-приложение для ведения коллекции сериалов и отслеживания просмотренных сезонов и серий. Пользователь может просматривать каталог сериалов, отмечать просмотренные эпизоды и хранить информацию о текущем прогрессе.

📺 Каталог сериалов

На сайте представлена коллекция популярных сериалов разных жанров.

✅ Отслеживание прогресса

Пользователь может отмечать просмотренные серии и сезоны.

🔎 Удобный поиск

Можно быстро находить сериалы по названию и фильтровать их по жанру.

📱 Адаптивный интерфейс

Сайт корректно отображается на компьютере, планшете и телефоне.

История создания проекта

Данный сайт был разработан в рамках учебной практики по дисциплине «Разработка веб-приложения на JavaScript». Основная цель проекта — создать удобное веб-приложение для работы с коллекцией сериалов.

  • Этап 1: разработка структуры страниц сайта
  • Этап 2: создание адаптивного интерфейса на HTML и CSS
  • Этап 3: добавление JavaScript для взаимодействия с пользователем
  • Этап 4: реализация отслеживания просмотренных серий и сезонов

Цель проекта

Цель проекта — создать простой и удобный сайт, который поможет пользователю вести список любимых сериалов, отмечать просмотренные серии и не терять прогресс просмотра.

Немного о сайте

Series Tracker — это учебный веб-проект, разработанный с использованием HTML5, CSS3 и JavaScript. Сайт включает главную страницу, каталог сериалов, страницу с информацией о проекте и страницу контактов.

JavaScript используется для интерактивности: фильтрации карточек, работы меню и отметки просмотренных серий.

Кратко о проекте

• Адаптивная верстка
