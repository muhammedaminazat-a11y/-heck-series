# 🎬 Serial Collection

**Веб-приложение для отслеживания просмотренных сериалов**

**Serial Collection** — это простое веб-приложение, разработанное в рамках учебной курсовой работы.  
Сайт создан с использованием **HTML5, CSS3 и Vanilla JavaScript**, включает адаптивную вёрстку, базовую логику и удобный интерфейс для учёта просмотренных серий.

---

## 🚀 Функционал проекта

### ✔️ Главная страница
- информационный блок о приложении;
- описание возможностей;
- простая навигация;
- адаптивный дизайн.

### ✔️ Трекер сериалов
- добавление сериалов;
- указание количества серий;
- учёт просмотренных серий;
- автоматический расчёт прогресса в процентах;
- обновление данных в реальном времени.

### ✔️ Работа с данными
- сохранение сериалов в `localStorage`;
- данные не теряются после перезагрузки страницы;
- возможность удаления сериалов;
- увеличение количества просмотренных серий.

### ✔️ Мобильное меню (бургер)
- открытие и закрытие меню;
- адаптация под телефон;
- простая навигация.

### ✔️ Дополнительные страницы
- главная;
- каталог;
- о проекте;
- контакты.

---
