🎬 Serial Collection

Веб-приложение для отслеживания просмотренных сериалов

Serial Collection — это простое веб-приложение, разработанное в рамках учебной курсовой работы. Сайт создан с использованием HTML5, CSS3 и Vanilla JavaScript, включает адаптивную верстку, базовую логику и удобный интерфейс для учета просмотренных серий.
